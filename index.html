yeezyhtml

by makakoa

Yeezy to read. Yeezy to write.

Yeezyhtml is a simple compiler which uses syntax inspired by o2.js to write static HTML and CSS.

Examples
  • ['a', {href: 'http://github.com'}, 'Github'] ---> <a href='http://github.com'>Github</a>
  • This page (source)
  • My resume + (source)

Compiling

node index exampleEntry index.htmlOutput
